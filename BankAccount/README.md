<h2>Bank Account Management System (Java)</h2>

 <h3>Project Overview</h3>

This is a simple Java-based mini project that demonstrates the concept of a Bank Account system using Object-Oriented Programming (OOP).<br>

The project allows basic operations like:<br>
--> Creating a bank account<br>
--> Displaying account details<br>
--> Managing balance

<h3> Features</h3>
--> Create a bank account with:<br>
    Account Number<br>
    Account Holder Name<br>
    Balance<br>
Display account details in a structured format<br>
Beginner-friendly implementation of Java classes and objects

<h3>Technologies Used</h3>
-> Java<br>
-> OOP Concepts (Class, Object, Constructor, Methods)
