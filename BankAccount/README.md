<h2>Bank Account Management System (Java)</h2>

 <h3>Project Overview</h3>

This is a simple Java-based mini project that demonstrates the concept of a Bank Account system using Object-Oriented Programming (OOP).

The project allows basic operations like:
Creating a bank account
Displaying account details
Managing balance

<h3> Features</h3>
--> Create a bank account with:
    Account Number
    Account Holder Name
    Balance
Display account details in a structured format
Beginner-friendly implementation of Java classes and objects

<h3>Technologies Used</h3>
-> Java
-> OOP Concepts (Class, Object, Constructor, Methods)
