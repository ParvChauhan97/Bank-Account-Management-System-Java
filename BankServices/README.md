# Bank Service Module (Core Java)

This module is a part of the **Bank Account Management System** project built using Core Java.  
It handles all the core banking operations and acts as the service layer between the user interface (Main Class) and the account data.

---

## 🚀 Features Implemented

-->  Create New Account  
-->  Deposit Money  
-->  Withdraw Money  
-->  Check Account Balance  
-->  Search Account by Account Number
